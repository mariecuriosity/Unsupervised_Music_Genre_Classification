This application is an unsupervised method of classifying music into genres based on Mel-Frequency Cepstrum Coefficients and SVMs

USAGE:
1) Run genFourierBeatsSegment.m
2) Run matlabsvm.m
3) Evaluation with evaluate_features_buildinsvm.m and matlabeval.m

Please note that the following dependencies are required:
Davies Beat Tracking:
M. E. P. Davies and M. D. Plumbley, “Context-Dependent Beat Tracking of Musical Audio”, IEEE Transactions on Audio, Speech and Language Processing, vol. 15, no. 3, 1009-1020, 2007.

Automatic Audio Segmentation: Segment Boundary and Structure Detection in Popular Music
E. Peiszer, “Ewald Peiszer: Automatic Audio Segmentation: Segment Boundary and Structure Detection in Popular Music”, Vienna University of Technology, August 2007.

VOICEBOX: Speech Processing Toolbox for MATLAB:
M. Brookes. “VOICEBOX: Speech Processing Toolbox for MATLAB”, Imperial College.
